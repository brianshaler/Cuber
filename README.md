Cuber
=====

Node.js CLI app to convert a spherical panorama source image into a cubic panorama

Install dependencies:

    npm install

Building app.js:

    coffee -j app.js -c src/app.coffee

Processing an image:

    node app.js --src images/demo.jpg --output output/
